## Typography Details

### Logo: "BREW & BEAN"
- Font Family: Lato, Helvetica Neue, Helvetica, Arial, sans-serif
- Font Weight: 900 (Heavy/Black)
- Color: #5D4037 (Coffee Brown)
- Size: Responsive clamp(1.5rem, 4vw, 2.5rem)

### Navigation Links: "HOME MENU ABOUT CONTACT"
- Font Family: Lato, Helvetica Neue, Helvetica, Arial, sans-serif
- Font Weight: 700 (Bold)
- Color: #5D4037 (Coffee Brown)
- Hover Color: #8D6E63 (Lighter Brown)
- Size: Responsive clamp(0.9rem, 2.5vw, 1.1rem)

### Call-to-Action Button: "Order Online"
- Font Family: Lato, Helvetica Neue, Helvetica, Arial, sans-serif
- Font Weight: 700 (Bold)
- Background: #5D4037 (Coffee Brown)
- Text Color: White
- Hover Background: #8D6E63 (Lighter Brown)
- Size: Responsive clamp(0.9rem, 2.5vw, 1rem)

## Visual Characteristics of Lato Font

Lato provides the perfect balance for a coffee shop brand:
- Clean and modern appearance
- Excellent readability at all sizes
- Professional yet approachable feel
- Versatile across different weights
- Web-optimized with WOFF2 format

The Heavy (900) weight gives the logo strong visual presence,
while the Bold (700) weight ensures navigation elements are clear
and actionable. The font's rounded terminals add warmth to the
brand without sacrificing professionalism.
